# Portfolio Website with Contact Form

This is a portfolio website that includes a fully functional contact form with validation and user feedback.

## Features
- Contact form with the following fields:
  - **First Name** (required)
  - **Last Name** (required)
  - **Email** (required)
  - **Subject** (required, dropdown with 3 options)
  - **Message** (required, minimum 20 characters)
  - **Phone Number** (optional)
- Real-time character counter for the message field.
- Validation for all required fields.
- Success message after form submission.
- Reset button to clear the form.

## Technologies Used
- **HTML**: For the structure of the website.
- **CSS**: For styling the website, including the contact form.
- **JavaScript**: For form validation and interactivity.
